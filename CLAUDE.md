# Project: PBX (Phone Home)

## Overview
PBX project on MX Linux (RAM-only live session).

## Git Conventions
- Always use **conventional commits** (`feat:`, `fix:`, `docs:`, `chore:`, `refactor:`, `style:`, `test:`, `ci:`, `build:`, `perf:`)
- Commit messages are enforced by commitlint via a git commit-msg hook
- Always commit and push to GitHub after completing a task

## Environment
- MX Linux live session (RAM-only, nothing persists across reboots)
- Default user: `demo`, sudo password: `demo`
- Node.js available, npm for package management
- Remote: git@github.com:roykollensvendsen/pbx.git
